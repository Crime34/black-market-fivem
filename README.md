# **Black Market – FiveM**

## **Présentation du projet**

Ce projet est un **système de Black Market pour serveur FiveM**, développé dans un cadre **personnel et éducatif**.
Il permet aux joueurs d’accéder à un **marché noir immersif** via une interface moderne, intuitive et optimisée pour le RolePlay.

Le système a été conçu pour offrir une **expérience fluide**, sécurisée et totalement intégrée à l’univers RP du serveur.

---

## **Objectifs**

* **Créer une vraie économie illégale immersive**
* **Centraliser l’achat d’objets du marché noir**
* **Sécuriser les transactions**
* **Offrir une interface moderne et optimisée**

---

## **Technologies utilisées**

* **Lua (FiveM)**
* **HTML / CSS**
* **JavaScript**
* **Framework FiveM** (ESX / QBCore / autre)

---

## **Fonctionnalités**

* **Interface Black Market moderne (tablette.)**
* **Système de catégories (armes, outils, protections, munitions, etc.)**
* **Achat d’objets en temps réel**
* **Affichage de l’argent du joueur**
* **Compatible RP sérieux**
* **Optimisation client / serveur**

---

## **Installation**

1. **Télécharger ou cloner le dépôt :**

```
git clone https://github.com/Crime34/black-market-fivem.git
```

2. **Placer le script dans le dossier `resources` de FiveM**

3. **Ajouter la ressource dans le `server.cfg`**

```
ensure black-market-fivem
```

4. **Configurer le fichier de configuration**

5. **Redémarrer le serveur**
